Resume the previously paused computer use execution.

1. Take a fresh screenshot to see the current screen state
2. Review what was completed before the pause
3. Continue from the next unfinished step or action
4. Follow the same verification and safety rules as the original execution

If the screen state has changed since the pause (user moved windows, closed apps, etc.),
adapt to the new state rather than assuming the old state is still valid.
