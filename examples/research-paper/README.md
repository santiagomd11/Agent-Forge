# Research Paper Workflow

A workflow for writing research papers with rigorous source verification.

## Quick Start

Read `agentic.md` and start

## Commands

| Command | Step | Description |
|---------|------|-------------|
| /start-paper | All | Full workflow: runs Steps 1-5 |
| /research-topic | 2 | Research phase with source verification |
| /create-outline | 3 | Generate structured outline |
| /write-sections | 4 | Write each section sequentially |
| /review-paper | 5 | Final review and assembly |
